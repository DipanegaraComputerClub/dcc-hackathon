# Dipanegara Computer Club FrontEnd

## Getting Started

1. Clone Repo 
`git clone https://github.com/DipanegaraComputerClub/dcc-hackathon.git;`

2. change directory
`cd frontend`

3. install package 
`bun install` || `pnpm install` || `npm install`

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
### Optional
use `--port=4001`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
